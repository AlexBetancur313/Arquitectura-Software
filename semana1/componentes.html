Componentes

Descripción

Hablaremos brevemente de algunos de estos modulos, cual es su función y en donde podemos encontralos.
Componentes de interfaz de usuario
La interfaz de usuario es el medio por el cual un usuario efectua control sobre una aplicación, dicha aplicación debera contar con controles graficos que puedan interactuar con el mouse y el teclado brindando asi un buena experencia de usuario. La aplicación también debera contar con ciertas caracteristicas que se aseguren de brindar esa buena experencia de usuario las cuales son atractivo visual, claridad, coherencia y flexibilidad.
Lógica de negocio
La logica de negocio es aquella en la que se toman decisiones basadas en situaciones reales del día día, cada aplicativo tiene su propia logica negocio que se basa en su modelo prensetado (CRM).
Acceso a bases de datos
Este componente es el que interactua directamente con la base de datos, tambíen hace operaciones con la base datos como lo es insertar datos, eliminar, actualizar y consultar datos.
Servicios web Son especificaciones tecnologicas que se basan en estandares a nivel global, que da como resultado un modelo de interacción de un sistema a otro sistema, algunos de estos estanderes son (XML,SOAP, URL O HTTP).
