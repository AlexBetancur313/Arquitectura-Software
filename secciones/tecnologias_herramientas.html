Tecnologías y Herramientas

Descripción

Cuando se va a iniciar con un nuevo proyecto o sistema, es sumamente importante saber cuales seran las herramientas y tecnologias con las que se van a trabajar, esto permite que desde un inicio se pueda tener un mayor control, porque dependiendo del proyecto se usa un lenguaje u otro, ya que no todos los lenguajes son igual de eficientes para todas las tareas, cada uno tiene su punto fuerte y si lo sabemos aprovechar desde un inicio, el proyecto puede ser de mejor calidad, escalabilidad, rendimiento y mantenimiento.

Ejemplos

  1. Lenguajes de Programación: Python, Java, Javascript, C++, Php, etc.
  2. Bases de Datos: MySQL, PostgreSQL, Oracle, MongoDB, etc.
  3. Frameworks: React, Angular, Django,Springboot, Laravel, etc
  4. Herramientas: Git, GitHub, Jenkins, Docker, PyTest, VS Code, etc.
